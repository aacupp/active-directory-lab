# Active Directory Lab Deployment

## Overview
Built a virtual lab simulating an enterprise AD environment using VirtualBox, Windows Server, and Windows 11.

## Tools Used
- VirtualBox
- Windows Server 2019
- Windows 11 Pro
- PowerShell
- Active Directory, DNS, Group Policy

## Key Features
- Domain controller setup with DNS
- Organizational Units and user creation
- Client VM joined to domain
- Group Policy deployment and testing
